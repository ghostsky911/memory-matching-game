### 1. HTML結構

- 創建網頁標題為 "MEMORY WATCHING GAME"
- 在網頁中添加主要區塊，包含12個可點擊的網格
- 在頁腳部分顯示遊戲狀態信息

### 2. CSS樣式

- 為所有元素設定字體為 Arial
- 為網格設定灰色背景和點擊指示符
- 網格區塊居中顯示，每個網格項目均勻分布

### 3. Javascript邏輯
互動性
玩家可以通過點擊不同的卡片來翻開它們，檢查是否有配對

- 圖片配對
- 玩家需要依次翻开两张卡片，如果两张卡片的图案相同，则这两张卡片会保持翻开状态；如果不同，则卡片会再次翻转至面朝下。- 
  -可用不同的圖案表示
  -張卡片隨機發
